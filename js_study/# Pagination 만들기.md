# Pagination 만들기

> 현재 페이지 번호와 총 페이지 수를 사용해서 특정 범위의 페이지 번호 목록을 출력하시오. (12 점)
>
> - ## 조건
>   - 출력되는 페이지 번호의 수는 11 개로 고정한다.  
>      단, 총 페이지 수가 11 개보다 작으면 총 페이지 수만큼 출력한다.
>   - 총 페이지 수가 11 보다 크고 현재 페이지 번호는 5 보다 크다고 가정했을 경우  
>      현재 페이지는 목록 가운데에 위치한다.
